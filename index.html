Sistema de Gestión Kiosko

Descripción del Proyecto

Sistema completo de punto de venta diseñado específicamente para kioscos y pequeños comercios. Incluye gestión de inventario en tiempo real, control de stock, módulo de proveedores, sistema de caja con múltiples formas de pago y generación de reportes detallados.

Características Principales

• Control de stock automático con alertas de productos agotados

• Sistema de códigos de barras integrado

• Módulo de proveedores con historial de compras

• Reportes de ventas diarias, semanales y mensuales

• Interface intuitiva y fácil de usar

• Backup automático de datos

Tecnologías Utilizadas

PHP MySQL JavaScript Bootstrap HTML5 CSS3
